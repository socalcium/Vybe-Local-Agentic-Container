What are Developer Tools?

This section provides development and debugging utilities for the Vybe application. Use these tools to monitor system performance, manage background jobs, and analyze application state.

  • System Monitor: View real-time system metrics like CPU, memory, and disk usage
  • Job Manager: Monitor and control background tasks and scheduled operations
  • Database Viewer: Inspect database tables and records for debugging
  • Log Viewer: Access application logs to troubleshoot issues

Note: These tools are intended for development and administrative purposes.
